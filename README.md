# vue-multi-page

## Project setup

```bash
yarn install
```

### Compiles and hot-reloads for development

```bash
# mobile
yarn run serve-mobile

# pc
yarn run serve-pc
```

### Compiles and minifies for production

```bash
# mobile
yarn run build-mobile

# pc
yarn run build-pc
```

### Lints and fixes files

```bash
yarn run lint
```
